description: AGENTS.md（CLAUDE.md）とコードベースの実態を比較し、陳腐化した記述を洗い出す。
---

## 手順

1. `/workspaces/sandbox/AGENTS.md` を読み込む

2. 以下の項目について、実際のファイル/ディレクトリ構成と照合する：
   - **プロジェクト構成**：記載されているディレクトリ・ファイルが実際に存在するか確認する（`find` や `ls` で検証）
   - **アーキテクチャ**：使用技術（フレームワーク、ライブラリ）のバージョンや種類が `package.json` 等と一致するか確認する
   - **コーディングルール**：ルールが現在のコードに反映されているか、代表的なファイルをいくつか読んで確認する
   - **作業ルール**：記載内容が現在の運用と合っているか確認する（例：`git log` でコミットメッセージのフォーマット確認）

3. 結果を以下のフォーマットで報告する：

```md
## AGENTS.md 陳腐化チェック結果

### 問題なし

- （一致していた項目を箇条書き）

### 要更新

- （ずれていた項目を箇条書き。実態と記述の差分を具体的に示す）

### 不明・要確認

- （判断できなかった項目）
```

4. 「要更新」の項目があれば、AGENTS.md の修正案を提示する（ユーザーの確認を取ってから実際に修正する）。
